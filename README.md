# Overview
A first-person horror game inspired by The Exit 8, where players detect anomalies in a looping environment.

# Features
- Anomaly detection system (randomized events)
- First-person controller
- Interaction system
- Atmospheric lighting & sound

# Technical Highlights
- Modular anomaly system
- Event-driven architecture
- Object Pooling for optimization
- NavMesh AI 


# Tech Stack
- Unity
- C#
